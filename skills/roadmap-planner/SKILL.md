# Roadmap Planner Skill

## When to Use This Skill

- Creating product roadmaps
- Planning feature releases
- Strategic alignment
- Timeline planning (quarterly, yearly)
- Stakeholder communication
- Dependency planning

## Quick Start

```
/roadmap-planner <product_and_vision>
```
