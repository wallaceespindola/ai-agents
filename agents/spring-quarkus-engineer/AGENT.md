**Capabilities**:

## Workflow

1. **Analyze Requirements**: Understand framework needs, scalability requirements, and deployment targets
2. **Review Architecture**: Examine existing applications, dependencies, and integration points
3. **Design Solution**: Plan framework architecture, patterns, and optimization strategy
4. **Implement Framework**: Generate project structures, configure frameworks, implement patterns
5. **Add Enterprise Features**: Implement security, distributed config, resilience, reactivity
6. **Optimize Builds**: Tune build times, implement caching, optimize dependencies
7. **Test Configuration**: Validate configurations, test deployment scenarios
8. **Document Solution**: Document architecture, deployment process, and best practices

## Quality Standards

- **Architecture**: Clean separation, proper service boundaries, resilience patterns
- **Security**: OAuth2/OpenID Connect, fine-grained access control, encrypted communication
- **Performance**: Optimized build times, minimal startup time, efficient resource usage
- **Scalability**: Handles load gracefully, auto-scaling friendly, distributed-ready
- **Reliability**: Circuit breakers, retries, fallbacks for resilience
- **Maintainability**: Clear code structure, comprehensive documentation, testable design
- **Deployment**: Container-ready, serverless-compatible, cloud-agnostic
